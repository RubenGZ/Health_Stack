🎯
Esta tarea la debe hacer Ruben manualmente antes de que Task 6 pueda probarse visualmente. El código ya está escrito — solo falta el modelo.
Buscar modelo en Sketchfab

Busca "low poly human anatomy muscles" → filtro CC-BY → necesitas meshes separados por músculo (no un solo mesh fusionado). Alternativa: Fiverr ~$30-50, solicitar modelo con meshes nombrados.

Abrir en Blender y renombrar meshes

Convención: mesh_pectoral_mayor_esternal, mesh_dorsal_ancho, etc. Sin sufijo _l/_r — AnatomyLens los añade automáticamente.

Exportar GLB con DRACO (<4MB)

File → Export → glTF 2.0 (.glb) · ✅ Compression: DRACO · ❌ Animations: off · Target <4MB

Colocar en frontend/models/anatomy_lens.glb

Verificar en gltf-viewer.donmccurdy.com que los meshes son separados y están nombrados correctamente.

// model.js — cómo se registran los meshes
gltf.scene.traverse((obj) => {
  const rawName = obj.name  // ej: "mesh_pectoral_mayor_esternal_l"
  const key = rawName
    .replace(/^mesh_/, '')          // → "pectoral_mayor_esternal_l"
    .replace(/_(l|r)$/, '')        // → "pectoral_mayor_esternal"
  registry.get(key).push(obj)  // ambos lados bajo la misma key
})

// Loader lazy — carga AnatomyLens solo la primera vez
let _lens = null
async function getLens() {
  if (_lens !== null) return _lens
  try {
    const mod = await import('./anatomyLens/index.js')
    _lens = mod.default
    await _lens.init(container)
  } catch (e) {
    _lens = false  // permanentemente SVG
  }
  return _lens
}

// selectExercise ahora es async
async function selectExercise(id) {
  // ... toggle, renderGrid, renderAffiliate ...
  const lens = await getLens()
  if (lens) {
    await lens.highlight(ex.id, ex.muscles)
    return  // AnatomyLens gestiona la leyenda
  }
  // Fallback SVG original — siempre disponible
  highlightMuscles(ex.muscles)
  renderLegend(ex.muscles)
}
